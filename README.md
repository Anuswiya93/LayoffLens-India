# LayoffLens India 🇮🇳

## India Tech Layoff Analytics & Prediction Dashboard

### Tools Used
- Python (Pandas, Matplotlib, Seaborn, Scikit-learn)
- MySQL
- Power BI
- Machine Learning (Random Forest)

### Key Insights
- 2025 had the highest layoffs in India — 20,302 employees
- TCS laid off 12,365 employees — highest in India
- Service companies dominated layoffs in 2025
- ML Model achieved 81.25% accuracy

### Project Structure
- Data cleaning and filtering in Python
- MySQL database for storage
- Power BI dashboard with 4 pages
- Random Forest ML model for prediction
